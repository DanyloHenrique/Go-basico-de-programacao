# Go - Básico de Programação

Este repositório contém uma coleção de arquivos de código-fonte em **Go (Golang)**, cobrindo conceitos básicos de programação. O objetivo deste projeto é fornecer exemplos de código que abordam tópicos essenciais, como variáveis, tipos de dados, estruturas de controle, funções e outras construções fundamentais da linguagem Go.

## Estrutura do Repositório

O repositório está organizado da seguinte forma:

- `1-helloWorld.go`: Exemplo simples de programa "Hello, World!".
- `2-variaveisTipos.go`: Demonstração de declaração e utilização de variáveis e tipos de dados.
- `3-condicionais.go`: Exemplos de estruturas condicionais (`if`, `else`, `switch`).
- `4-repeticao.go`: Exemplos de estruturas de repetição (`for`, `while`, `range`).
- `5-listas.go`: Trabalhando com arrays e slices.
- `6-maps.go`: Exemplo de uso de mapas (dicionários).
- `7-struct.go`: Definição e manipulação de estruturas (structs).
- `8-ponteiros.go`: Manipulação de ponteiros.
- `9-goroutines.go`: Exemplos de uso de goroutines para programação concorrente.
